钛盘 AI 客服知识库 02：文件列表

文档角色
- 本文件覆盖文件列表页面（/?tmpui_page=/vx&module=filelist）的全部功能。
- 包括上传（含 CLI）、文件夹管理、文件操作、分享、相册视图、摆摊出售、视频在线观看。

命中条件
- 用户提到上传、下载、文件夹、新建文件夹、重命名、删除、有效期、相册、相册模式、分享文件、分享文件夹、摆摊、出售文件、设定售价、取消出售、CLI 上传、钛盘上传工具、tmplink、tmplink-cli、在线观看、视频播放。

排除条件
- 用户明确提到直链、直链 API、API Key、自定义域名、CNAME 时，转到 05 直链。
- 用户问的是文件下载页（/file?ukey=...）上的操作，转到 03 文件下载页。

页面入口
- 文件列表：/?tmpui_page=/vx&module=filelist

========== 上传 ==========

网页上传
- 点击工具栏上传按钮（云形上传图标）或将文件拖入页面区域即可触发上传。
- 支持分片上传，默认分片大小 80 MB，默认最多同时上传 10 个文件。
- 单文件内部串行分片，不同文件之间可并发。
- 单文件大小上限 50 GB。
- 上传前端自动选择可用上传服务器，本地保存的可用服务器会优先沿用。
- 非赞助者通常不能手动切换上传服务器。
- 赞助者可在设定页调整分片大小和同时上传文件数（详见 07 设定）。
- 新设置只影响之后新加入队列的任务，不影响已在队列中的任务。

CLI 上传（钛盘上传工具）
- 官方仓库：https://github.com/tmplink/tmplink_uploader
- 工具包含两个程序：tmplink（交互式，适合新手）和 tmplink-cli（命令行/脚本，适合进阶用户）。
- 获取 Token 路径：
  1. 登录钛盘，进入文件列表。
  2. 点击工具栏的上传按钮，弹出上传面板。
  3. 点击「重新设定」，在「命令行上传」区域复制 Token。
- 首次使用 tmplink-cli：
  1. 执行 tmplink-cli -set-token 你的Token  保存 Token。
  2. 执行 tmplink-cli -file 文件路径  上传文件。
- 上传问题排查：先检查 Token 是否有效、文件是否超限、网络是否中断；可用 tmplink-cli -debug 查看详细日志。
- 不要把 Token 分享给他人，Token 等同于当前账号的访问凭证。

========== 文件夹管理 ==========

新建文件夹
- 点击工具栏「新建文件夹」按钮（文件夹加号图标）。
- 弹窗中填写文件夹名称，可选择是否启用「私有文件夹」开关。
- 私有文件夹：仅自己可见，不能被他人访问或分享。
- 子文件夹创建时，私有设置不可单独设置，继承父级。
- 点击「创建」完成。

文件夹权限设置（进入文件夹后）
- 进入某个文件夹后，左侧边栏会出现「文件夹权限」区域。
- 「私有文件夹」开关：开启后文件夹变为私有，关闭后恢复公开。
- 「允许被搜索」开关：开启后其他用户可通过搜索引擎找到该文件夹中的文件。
- 这两项设置只有文件夹拥有者可以操作。

重命名文件夹
- 点击文件夹行右侧的「更多」按钮 → 选择「重命名」 → 输入新名称 → 确认。

删除文件夹
- 点击文件夹行右侧「更多」按钮 → 选择「删除」。
- 桌面不能分享，也不能删除。

分享文件夹
- 只有公开（非私有）的文件夹才能分享。
- 进入文件夹后，点击工具栏的分享按钮，或点击文件夹行上的分享图标，复制分享链接。
- 收到链接的人可以浏览该文件夹内的文件并下载。
- 桌面不能分享。

========== 文件操作 ==========

打开文件页
- 点击文件名，在新标签页打开文件下载页（/file?ukey=...）。

下载文件
- 点击文件行右侧的下载图标按钮，即可下载。
- 也可进入文件下载页后操作。

复制分享链接
- 点击文件行右侧的分享图标按钮，复制文件分享链接（格式：/f/文件标识）。
- 分享链接可直接发给他人，对方打开即可下载。

右键菜单（文件）
- 在文件行上右键单击，弹出快捷菜单，可执行以下操作：
  - 打开：在新标签页打开文件下载页。
  - 下载：直接下载文件。
  - 出售：设定该文件的售价（赞助者专属，见「摆摊出售」章节）。
  - 取消出售：移除已设定的售价（红色，仅已设价时显示）。
  - 重命名：修改文件名。
  - 通过直链分享：将文件加入直链分享（需先在直链模块绑定域名）。
  - 在线观看：视频文件在线播放（赞助者专属，见「在线观看」章节）。
  - 修改有效期：永久保存 / 24 小时 / 3 天 / 7 天。
  - 删除：删除文件（红色）。

修改有效期
- 右键文件 → 修改有效期 → 选择永久保存、24 小时、3 天或 7 天。
- 修改后新的有效期立即生效。

========== 相册视图 ==========

切换到相册视图
- 工具栏右侧有视图切换按钮组：列表图标和相册图标。
- 点击相册图标（图片/照片图标）切换到相册模式。
- 移动端：点击顶部工具栏右侧的图片图标。

相册模式特性
- 相册模式只展示当前目录中的图片文件，文档、压缩包、视频等非图片文件不显示。
- 如果目录中没有图片，相册显示为空。
- 相册模式下可调整网格大小（普通/小图）。
- 支持下载全部照片、下载选中照片。
- 点击图片可进入灯箱浏览模式。

退出相册视图
- 点击工具栏左侧的列表图标，切换回普通列表视图。

========== 摆摊出售 ==========

前置条件
- 摆摊出售是赞助者专属功能，非赞助者操作会提示「此功能需要赞助者权益」。
- 只能对自己仓库中的文件设定售价。

设定售价
1. 进入文件列表，找到目标文件，右键单击。
2. 菜单中选择「出售」。
3. 弹出设定售价窗口，输入售价（单位：点数，范围 1 至 10000）。
4. 确认后，文件列表中该文件会显示价格标签（金色点数图标 + 数字）。

取消出售
1. 右键单击已设定售价的文件。
2. 菜单中选择「取消出售」（红色显示）。
3. 确认后价格标签移除，文件恢复为普通文件。

调整价格
- 对已出售的文件再次右键选择「出售」，可重新输入价格覆盖原售价。

查看出售记录
- 出售记录在点数页管理：/?tmpui_page=/vx&module=points → 左侧「出售文件」标签。

买家如何购买
- 买家在文件列表中看到「未购买」标记的付费文件时，点击后弹出购买确认窗口。
- 确认后系统扣除相应点数，购买成功即可正常下载。
- 若买家点数不足，页面会提示点数不足，需先充值（见 06 商城与点数）。

========== 在线观看（视频） ==========

前置条件
- 在线观看是赞助者专属功能。
- 仅对视频类文件显示该选项。

操作路径
1. 右键单击视频文件，菜单中选择「在线观看」。
2. 弹出播放器选择：
   - Browser：在浏览器内直接播放。
   - PotPlayer：用 PotPlayer 播放（需本地已安装）。
   - IINA：用 IINA 播放（macOS，需本地已安装）。
   - nPlayer：用 nPlayer 播放（需本地已安装）。
   - Stream URL：复制流媒体链接，供其他播放器使用。

========== 标准回答流程 ==========

- 用户问「怎么上传文件」：先回答网页上传入口，按需补充 CLI 上传工具。
- 用户问「怎么命令行上传」：回答钛盘上传工具，区分 tmplink 和 tmplink-cli，给出 Token 获取步骤。
- 用户问「怎么分享文件给别人」：回答复制分享链接按钮，说明收到链接的人可在文件下载页下载。
- 用户问「怎么分享文件夹」：说明文件夹需为公开状态，点击分享图标复制链接。
- 用户问「怎么设定摆摊/出售文件」：说明前置条件（赞助者），再给出右键→出售的步骤。
- 用户问「为什么看不到出售按钮」：先检查是否为赞助者，再确认是否为文件拥有者。
- 用户问「相册里为什么看不到文档/压缩包」：说明相册只显示图片文件。
- 用户问「怎么改文件有效期」：右键文件→修改有效期。

========== 禁止补充 ==========

- 不要主动补充直链 API Key、自定义域名、CNAME、优先配额，除非用户明确追问直链。
- 不要把普通 CLI 上传说成直链 API 接入。
