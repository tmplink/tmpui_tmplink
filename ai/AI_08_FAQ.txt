钛盘 AI 客服知识库 08：常见问题

说明
- 本文件用于高频问法速查。
- 回答时优先参考对应主题文档，再用这里校准口径。

文件列表
- CLI 上传：使用官方钛盘上传工具 https://github.com/tmplink/tmplink_uploader；tmplink 适合交互式使用，tmplink-cli 适合脚本；Token 获取路径为“文件列表 -> 上传按钮 -> 重新设定 -> 命令行上传”；首次使用先执行 tmplink-cli -set-token 你的Token，再执行 tmplink-cli -file 文件路径；不要主动提直链 API、域名绑定或 CNAME。
- 分享文件：点击文件行右侧分享图标复制普通分享链接；对方会在文件下载页下载；不要默认回答直链。
- 分享文件夹：文件夹必须公开；进入文件夹后点击工具栏分享按钮或文件夹行分享图标复制链接；桌面不能分享。
- 相册里为什么看不到全部文件：相册只显示图片文件；要看全部文件，切回普通列表视图。
- 摆摊出售：先确认用户是否为赞助者；右键文件 -> 出售 -> 输入 1 到 10000 点数；取消出售则右键文件 -> 取消出售。
- 为什么没有出售按钮：只有赞助者且文件拥有者本人才能看到并使用出售功能。
- 修改文件有效期：右键文件 -> 修改有效期 -> 选择永久保存、24 小时、3 天或 7 天。

文件下载页
- 命令行下载：在文件下载页点“下载选项”，选择“复制 curl 命令”或“复制 wget 命令”。
- 点击下载要求购买：说明该文件已设售价，需要点数购买；确认窗口会要求等待 5 秒；购买后文件加入桌面，默认保留 7 天；点数不足时需先充值。
- 下载太慢：引导到“提高下载速度”，三种方式分别是开启浏览器并行下载、使用 IDM 或 Motrix 等多线程工具、购买赞助者权益。

密记
- 看不到密记内容：先判断是否未设置密钥，再判断密钥是否错误；只有密钥正确且已设置时才能正常显示。
- 刷新后为什么还要输入密钥：正常刷新不会丢，密钥保存在浏览器本地；退出登录后才会清除，需要重新设置。
- 平台能不能看到密记：不能，平台只能看到浏览器侧 AES 加密后的密文。

直链
- 什么时候需要用直链：只有在用户有自己的域名、要通过自定义域名分发文件、要做流量统计或直链 API 接入时才用；普通上传下载和普通分享都不需要直链。
- 普通 CLI 上传和直链 CLI 或 API 是否一样：不是；普通 CLI 上传用文件列表里的 Token，直链 API 用直链页面里的独立 API Key；用户只说 CLI 上传时，默认按普通上传工具回答。
- 直链为什么用不了：优先检查域名是否已绑定、CNAME 是否正确指向 cdn.5t-cdn.com、DNS 是否已生效。

商城与点数
- 赞助者价格：普通赞助者 1 个月 6 元，1 年 72 元；初次赞助特典限一次，1 年 36 元；也可用 100 分享值兑换 1 个月，每月一次。
- 私有空间价格：256 GB 6 元/月，72 元/年；1 TB 18 元/月，216 元/年；3 TB 66 元/月，792 元/年；5 TB 120 元/月，1440 元/年。
- 点数充值：支付宝 1 元 = 100 点，单次最少 1 元、最多 500 元；PayPal 1 USD = 600 点，单次最少 5 美元、最多 500 美元；入口为 /?tmpui_page=/vx&module=points。
- 积分商城：京东 E 卡 50 元 / 6500 点，100 元 / 13000 点，500 元 / 65000 点；可能售罄；点数不足时会提示先充值。
- 优先配额是什么意思：界面名称保持“优先配额”，实际含义是直链流量配额；价格在商城查看；配额不失效且可叠加。

设定
- 分片大小和同时上传数怎么改：这是赞助者专属功能；进入设定页 -> 效率设置修改；自动保存，只影响后续新进队列任务。
- 复制链接格式怎么改：进入设定页 -> 偏好设置 -> 复制风格，切换纯链接、文件名 + 链接或 Markdown 风格。

到期行为
- 通用规则：私有空间不足时，系统会自动删除最近上传到私有空间的文件，直到总量回到可用容量；私有空间降到 0 时，私有空间文件会全部删除。
- 赞助者到期：高速通道、在线视频、效率设置、摆摊、优先认证、头像签名展示等权益立即失效；附带的 100 GB 私有空间同时取消；如果没有单独购买存储包，私有空间会降到 0；建议到期前备份或续费。
- 存储包到期：对应容量从总私有空间中扣除，之后按通用规则处理文件；不同容量续费可叠加容量，相同容量续费只延长有效期。
