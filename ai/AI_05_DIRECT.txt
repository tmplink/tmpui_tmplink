钛盘 AI 客服知识库 05：直链

文档角色
- 本文件覆盖直链模块（/?tmpui_page=/vx&module=direct）的全部功能。
- 直链是独立于普通分享的一套文件分发体系，需要用户拥有自己的域名。
- 只有用户明确命中直链关键词时才使用本文件。

命中条件
- 用户明确提到：直链、直链分享、直链 API、API Key、自定义域名、绑定域名、CNAME、优先配额（功能性问题）、直链流量、直链统计。

排除条件
- 用户只问普通上传下载、CLI 上传下载、普通分享、文件夹分享、相册、密记时，不使用本文件。

页面入口
- 直链：/?tmpui_page=/vx&module=direct

========== 直链是什么 ==========

- 直链是独立于普通分享的文件分发体系。
- 用户可以把仓库中的文件加入直链，通过自定义域名对外提供下载。
- 同一个文件可以创建多份直链分享，已创建的直链分享可以撤销。
- 直链支持流量统计和下载统计。

========== 前置条件 ==========

- 直链功能需要用户拥有自己的域名。
- 必须先完成域名绑定，才能使用直链的其他功能。
- 如果 CNAME 未正确配置，直链相关页面会提示域名解析异常。

========== 域名绑定 ==========

CNAME 配置（先在域名服务商操作）
1. 登录你的域名 DNS 服务商（如阿里云、Cloudflare 等）。
2. 为要使用的域名添加一条 CNAME 记录，指向：cdn.5t-cdn.com
3. 等待 DNS 生效（通常几分钟到数小时不等）。

在钛盘直链页面绑定域名
1. 进入直链页面：/?tmpui_page=/vx&module=direct
2. 在域名输入框中填写你的域名（例如 files.example.com）。
3. 点击绑定，系统校验 CNAME 是否正确指向。
4. 绑定结果说明：
   - 绑定成功：域名可以正常使用。
   - CNAME 未配置：需要先在域名服务商添加 CNAME 记录。
   - 域名已被使用：该域名已被其他账号绑定。
   - 域名不合法：域名格式错误，检查输入是否正确。
   - 绑定失败：服务异常，稍后重试。

========== API Key ==========

- 直链页面有独立的 API Key，用于 CLI 脚本接入和 API 调用。
- API Key 与普通文件列表的 Token 不同，二者相互独立。
- 只在用户明确说要接直链 API、使用直链 CLI、进行脚本接入时，才展开 API Key 内容。

========== 直链文件管理 ==========

- 在直链页面可以管理已加入直链的文件和文件夹。
- 支持直链文件管理、直链文件夹管理。
- 可撤销已创建的直链分享。

文件夹直链（在文件列表中操作）
- 在文件列表中进入某个文件夹后，左侧边栏会出现「直链」区域。
- 启用「允许文件夹直链」开关，该文件夹内的文件可通过直链域名访问。
- 需要先在直链模块绑定域名，否则开关会提示「请先在直链模块绑定域名」。

========== 优先配额 ==========

- 界面上当前显示名为「优先配额」，该名称暂不调整。
- 对外解释口径：优先配额准确来说是直链流量配额，用于通过直链向他人提供下载服务时消耗。
- 优先配额不会过期，多次购买会叠加。
- 购买入口：/?tmpui_page=/vx&module=shop（见 06 商城与点数）。
- 如果用户只问优先配额价格，进入 06 商城与点数回答，不展开直链接入说明。

========== 标准回答流程 ==========

- 用户问「怎么绑定直链域名」：先说明需要在域名服务商配置 CNAME 指向 cdn.5t-cdn.com，再说明在直链页面填写域名绑定。
- 用户问「CNAME 怎么配」：说明在域名 DNS 服务商添加 CNAME 记录，目标为 cdn.5t-cdn.com。
- 用户问「直链为什么不能用」：优先检查域名是否已绑定、CNAME 是否正确指向。
- 用户问「怎么用 API 上传到直链」：先说明 API Key 获取，再说明直链 API 接入方式。
- 用户问「优先配额多少钱」：只报价格，不展开 API 接入，详见 06 商城与点数。

========== 禁止混答 ==========

- 不要把普通分享说成直链。
- 不要把普通 CLI 上传（钛盘上传工具）说成直链 API。
- 用户没提直链时，不要主动把普通问题带入本文件。
