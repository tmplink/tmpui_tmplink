接口文档（服务端 API）
以下内容迁移自旧版直链 API 文档，便于对接脚本/CLI。
POST https://tmp-cli.vx-cdn.com/app/upload_cli（上传文件）
Endpoint：https://tmp-cli.vx-cdn.com/app/upload_cli
Content-Type：application/x-www-form-urlencoded
示例
curl -k -F "file=@/root/test.bin" -F "key=API key" -X POST "https://tmp-cli.vx-cdn.com/app/upload_cli"
参数
key API Key
mrid 文件有效期
model 存储模式
99：永久
0：24h 后销毁
1：3 天
2：7 天
返回值
Status：
1：上传完成，此时返回的 data 为文件的 UKEY。
0：上传失败，无效的请求。
2：上传失败，文件大小超过限制。
3：上传失败，服务器繁忙，无法处理此次请求。
4：上传失败，私有空间不足，无法存储文件。
5：上传失败，目前上传的文件总量已经超出了当日配额（存储至私有空间时不受限制）。
6：上传失败，API Key 已失效。
Data：成功时为 UKEY；失败时为错误信息。
POST https://tmp-api.vx-cdn.com/services/direct（通用说明）
Endpoint：https://tmp-api.vx-cdn.com/services/direct
Content-Type：application/x-www-form-urlencoded
通用参数：action、key（API Key）、page。
POST https://tmp-api.vx-cdn.com/services/direct（action=quota 获取直链流量配额）
参数
actionquota
keyAPI Key
返回值
status：1 有数据；0 没有数据
data：quota（已购买的配额）
POST https://tmp-api.vx-cdn.com/services/direct（action=list_of_workspace 获取仓库文件列表）
参数
actionlist_of_workspace
keyAPI Key
page页码
返回值
status：1 有数据；0 没有数据
data：（Array）每项包含 ukey / name / size
POST https://tmp-api.vx-cdn.com/services/direct（action=list_of_direct 获取直链文件列表）
参数
actionlist_of_direct
keyAPI Key
page页码
返回值
status：1 有数据；0 没有数据
data：（Array）每项包含 dkey / link / name / size / etime
POST https://tmp-api.vx-cdn.com/services/direct（action=link_add 添加直链）
参数
actionlink_add
keyAPI Key
ukeyUKEY
valid_time有效期（分）
download_limit下载次数限制
返回值
status：
1：完成
0：失败
1001：失败，文件不存在
1002：失败，内部错误
1005：失败，文件尚未就绪，请稍后再添加
POST https://tmp-api.vx-cdn.com/services/direct（action=link_del 移除直链）
参数
actionlink_del
keyAPI Key
dkeyDKEY
delete同时删除文件
返回值
status：
1：完成
0：失败
1003：失败，没有找到对应的直链
